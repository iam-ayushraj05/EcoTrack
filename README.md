# EcoTrack

EcoTrack is a carbon footprint tracker application that enables users to monitor their personal carbon emissions and learn how to reduce them.

## 🚀 Quick Start

1. **Install Dependencies**:
   ```bash
   npm install
   cd backend
   npm install
   ```

2. **Start the Application**:
   From the root directory:
   ```bash
   npm start
   ```

3. **Open the App**:
   Navigate to `http://localhost:3000` in your web browser.
